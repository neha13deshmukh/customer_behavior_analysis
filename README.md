# Customer Behavior Analysis Dashboard

## Overview
This project analyzes customer shopping behavior using Python, MySQL, and Power BI. It includes data cleaning, Exploratory Data Analysis (EDA), SQL analysis, and dashboard creation to generate business insights.

---

## Tools & Technologies
- Python
- MySQL
- Power BI

---

## Project Workflow
- Loaded and cleaned data using Python
- Performed EDA to identify trends and patterns
- Executed SQL queries in MySQL for business insights
- Built an interactive Power BI dashboard

---

## Dashboard Insights
- Total Customers: 3900
- Average Order Value: $59.76
- Average Review Rating: 3.75
- Revenue trends by season
- Product category performance
- Subscription vs Non-subscription analysis

## 🛠️ How to Use This Project

### Clone the Repository
```bash
git clone https://github.com/neha13deshmukh/customer_behavior_analysis.git
cd customer_behavior_analysis
```

### Install Required Libraries
```bash
pip install pandas numpy matplotlib seaborn mysql-connector-python
```

### Open Jupyter Notebook
Run:
```bash
jupyter notebook
```

Open:
`Customer_shopping_cleaned.ipynb`

The notebook includes:
- Data Import
- Data Exploration
- Data Cleaning
- SQL Database Connection
- Data Loading into MySQL
